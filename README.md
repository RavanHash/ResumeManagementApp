# ResumeManagementApp
A personal project to better understand full-stack application development with a backend in .NET and a frontend in React.


## Technologies
-  Asp.Net Core (.NET 7.0)
-  React 18
-  TypeScript
-  MS SQL SERVER
-  Entity Framework Core
